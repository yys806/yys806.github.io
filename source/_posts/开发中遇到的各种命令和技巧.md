# 六、服务器问题

目前我是购入了一个海外的服务器，$14.99一年，用来部署我的机器人。

其中，需要让我的窗口固定，不随我的远程控制窗口的关闭而关闭，用到的是screen这个包，主要是以下几个命令：
(1)列出所有的会话
```
screen -ls 
```
(2)创建会话
```
screen -S 会话名
```
(3)离开会话
```
Ctrl+A, D
```
(4)恢复会话
```
screen -r 会话名 
```
(5)终止会话
```
screen -X -S 会话名 quit 
```
