6. 我给自己的安全提醒（重要）

我在 opencode.json 里发现了明文敏感信息（例如部分 MCP token/header）。

我应该立即做三件事：

  1. 把 token 改成环境变量注入，不再在 JSON 明文保存。
  2. 轮换已暴露凭证（至少 GitHub 和 Stripe 相关凭证）。
  3. 给配置文件增加本地访问控制，避免同步到公开仓库。
